# Pedra, Papel, Tesoura, Lagarto e Spock

## Descrição do Projeto

Este projeto consiste em um jogo interativo desenvolvido com HTML, CSS e JavaScript baseado na versão expandida do clássico Pedra, Papel e Tesoura.

Além das opções tradicionais, o jogo inclui:
- Lagarto  
- Spock  

O sistema permite que o usuário escolha uma opção enquanto o computador gera uma escolha aleatória, exibindo o resultado da rodada e atualizando a pontuação do jogador.

---

## Objetivo

O objetivo do projeto é praticar conceitos de desenvolvimento web, lógica de programação e manipulação do DOM através da criação de um jogo dinâmico e interativo.

---

## Tecnologias Utilizadas

- HTML
- CSS  
- JavaScript  

---

## Funcionalidades

- Escolha entre:
  - Pedra  
  - Papel  
  - Tesoura  
  - Lagarto  
  - Spock  

- Sistema de pontuação  
- Resultado automático da partida  
- Escolha aleatória do computador  
- Modal de regras do jogo  
- Botão de reinício da partida

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/pedra-papel-tesoura-lagarto-spock.git
```

2. Acesse a pasta do projeto:
```bash
cd pedra-papel-tesoura-lagarto-spock
```

3. Abra o arquivo:
```bash
index.html
```

---

## Funcionamento do Jogo

1. O jogador escolhe uma das opções disponíveis  
2. O computador gera uma escolha aleatória  
3. O sistema compara as escolhas  
4. O resultado da rodada é exibido:
   - Vitória  
   - Derrota  
   - Empate  

5. A pontuação é atualizada automaticamente  

---

##  Diferenciais
